Tutorial

🏠 Home

Your dashboard showing pending tasks, notes count, and last opened board. Quick access to create new tasks and notes.

✓ To-do

Manage your tasks with categories (School, Personal, Other). Set due dates and mark tasks as complete. Click a task to edit or delete it.

📅 Calendar

View your month and click any day to add tasks specific to that date. Navigate between months using the arrows.

📝 Notes

Create and organize notes into folders. Use the search bar to find notes quickly. Click any note to edit it.

🎨 Board

A digital whiteboard for drawing, adding text, sticky notes, and images. Create multiple boards and export them as PNG images.

💾 Data Storage

All your data is saved locally on this device using localStorage. No account or internet required!
